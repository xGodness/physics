Problem #3.2

Определить напряженность электрического поля, потенциал которого зависит от координат \( x \), \( y\ \) по закону \( \varphi(x, y) \).

Given:

\( a) \ \varphi = a (x^2 - y^2)\)

\( b) \ \varphi = axy \)

Answer:

\( a) \ \overrightarrow{E} = -2ax\cdot\vec{i} + 2ay\cdot\vec{j} \)

\( b) \ \overrightarrow{E} = -ay\cdot\vec{i} - ax\cdot\vec{j} \)

Problem #3.4

Найти потенциал электрических полей.

Given:

\( a) \ \overrightarrow{E} = a(y\cdot\vec{i}\ + x\cdot\vec{j}\ ) \)

\( b) \ \overrightarrow{E} = 2axy\cdot\vec{i} + a(x^2 - y^2)\cdot\vec{j} \)

Answer:

\( a) \ \varphi = -axy \)

\( b) \ \varphi = -ax^2y + {1 \over 3} ay^3 \)

Result
